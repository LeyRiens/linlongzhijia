# task_remote:
开启uart1的dma中断接收，数据接收使用蓝牙模块，波特率9600，双缓冲区交替接收，DMA和CPU写入读取，避免数据覆盖

解析后并过更改全局变量，分发指令给其他任务，并交换缓冲区

# task_chassis:
接收指令并设置对应x,y,w的目标量
通过计算x+y+w*(s+t)/2解算底盘各个电机目标速度

判断各个电机目标速度的正负来设置电机转动方向

通过霍尔编码器读取速度，进行pid闭环计算最终电机的输出（因为引脚不够暂时没有实现）

最后设置占空比控制实际转速

# task_arm:
因为抓取目标的位置相对固定以及sg90舵机精度较差，所以使用读取固定轨迹表。云台计划用电机。

因为设计小车主要依靠滚筒收集小球和方块，所以机械臂的代码还没写
